## [234. Palindrome Linked List](https://leetcode-cn.com/problems/palindrome-linked-list/)

## 1. 题目描述(简单)

Given a singly linked list, determine if it is a palindrome.

Example 1:
```
Input: 1->2
Output: false
```
Example 2:
```
Input: 1->2->2->1
Output: true
```
**Follow up:**
Could you do it in O(n) time and O(1) space?
